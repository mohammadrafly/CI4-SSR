## Getting Started

# CI4-SSR
Ikuti perintah berikut biar CI nya bisa lari

### Built With

* CI4

### Prerequisites

Penting! install bahan dibawah ini:
* composer
* decent brain 

### Installation

* Install Dependencies
   ```sh
   composer install
   ```
* Migrate Database
   ```sh
   php spark migrate
   ```
* Run App
   ```sh
   php spark serve
   ```
